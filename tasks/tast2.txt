# Эпизод 2: Эксплуатация

Цель: Взломать небезопасную конфигурацию

Команда для запуска:

docker run -it --rm \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v /:/host \
  --privileged \
  vulnerable-app:latest

  Задания: 
  1. Создать Dockerfile и поместить команду 
  2. Зпустить контейнер и проверить работу
  3. Предоставить флаг который был "украден"
